# Gerenciador de Tarefas

App gerenciador de tarefas (todo's) desenvolvido em **React**.

Conceitos e ferramentas utilizadas:

- **React**;
- **Typescript**;
- **Tailwind**;
- **useReducer**;
- Hook Customizado **useLocalStorage**;
- **React Hot Toast**;
